# rursache's Homebrew Tap

This repository hosts custom Homebrew formulas for my CLI tools and projects.

## Installation

Add the tap:

```bash
brew tap rursache/tap
```

## Available Formulas

| Formula Name | Description |
|---------|-------------|
| [`solo-cli`](https://github.com/rursache/solo-cli) | A terminal-based user interface for SOLO.ro, an online accounting platform for Romanian freelancers (PFA) |
| [`loto-cli`](https://github.com/rursache/loto-cli) | A terminal-based interface for Loteria Romana, Romania's national lottery |
| [`reddit-cli`](https://github.com/rursache/reddit-cli) | A quick CLI to read Reddit posts, no API key or auth required |
| [`watchwise-cli`](https://github.com/rursache/watchwise-cli) | Smart movie & TV show recommender powered by your IMDB ratings |
| [`bird-cli`](https://github.com/rursache/bird-bin/releases) | A fast X/Twitter CLI for tweeting, replying, and reading via GraphQL |
| [`insert_dylib`](https://github.com/tyilo/insert_dylib) | CLI for injecting a dylib load command into a Mach-O binary (iOS, macOS, etc) |

## Usage

Install any formula from this tap:

```bash
brew install rursache/tap/<formula>
```

## Updating

Update the tap and upgrade installed formulas:

```bash
brew update
brew upgrade
```
